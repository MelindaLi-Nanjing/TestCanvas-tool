Markup Help

  • Select an item by clicking it, then drag to move it.
  • Resize with corner handles. Text items use the side handle to change width and wrap.
  • Items snap to image center lines and nearby item edges while dragging.
  • Use callouts for numbered steps with optional explanatory text.
  • Shortcuts: P pen, A arrow, R rectangle, H highlight, T text, C callout, Delete remove, Ctrl+Z undo, Ctrl+C copy, Ctrl+V paste, Ctrl+D duplicate, F1 help.
